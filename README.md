S. F. Ung, A. Mahajan, and D. R. Reichman, _Study of the Triangular-Lattice Hubbard Model with Constrained-Path Quantum Monte Carlo._ [arXiv:2603.14808](https://arxiv.org/abs/2603.14808v1)

The AFQMC calculations were performed using a development version of the `trot` Python package: [https://github.com/ankit76/trot/tree/legacy/main-2026-04-04](https://github.com/ankit76/trot/tree/legacy/main-2026-04-04).
